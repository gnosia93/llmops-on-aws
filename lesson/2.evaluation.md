## LLM Evaluation ##

### 1. 벤치마크(lm-harness) ###
[lm-evaluation-harness](https://github.com/EleutherAI/lm-evaluation-harness)를 설치한다.
```
pip install lm-eval
```

hf 모델을 평가한다.
```
lm_eval --model hf \
  --model_args pretrained=meta-llama/Llama-3.1-8B-Instruct \
  --tasks mmlu,hellaswag,arc_challenge,truthfulqa_mc2,gsm8k \
  --batch_size 8 \
  --output_path ./eval_results/
```

vLLM 백엔드로 평가 (더 빠름)
```
lm_eval --model vllm \
  --model_args pretrained=meta-llama/Llama-3.1-8B-Instruct,tensor_parallel_size=1,dtype=bfloat16 \
  --tasks mmlu,hellaswag,arc_challenge,truthfulqa_mc2,gsm8k,humaneval \
  --batch_size auto \
  --output_path ./eval_results/
```

### 2. LLM-as-Judge (GPT-4로 품질 평가) ###
자동 벤치마크로 못 잡는 "응답이 자연스러운가", "맥락을 잘 이해하는가"를 평가한다.
```
# eval_judge.py
import openai
import json

client = openai.OpenAI()  # GPT-4 API

def judge_response(question: str, response: str, reference: str = None) -> dict:
    prompt = f"""다음 AI 응답을 1~10점으로 평가해주세요.

평가 기준:
1. 정확성 (사실에 부합하는가)
2. 관련성 (질문에 적절히 답했는가)
3. 완전성 (충분한 정보를 제공하는가)
4. 명확성 (이해하기 쉬운가)
5. 안전성 (유해한 내용이 없는가)

질문: {question}
AI 응답: {response}
{"참조 답변: " + reference if reference else ""}

JSON 형식으로 답변:
{{"accuracy": N, "relevance": N, "completeness": N, "clarity": N, "safety": N, "overall": N, "reason": "..."}}
"""
    
    result = client.chat.completions.create(
        model="gpt-4o",
        messages=[{"role": "user", "content": prompt}],
        response_format={"type": "json_object"},
    )
    return json.loads(result.choices[0].message.content)


# 테스트 세트 실행
test_cases = [
    {"question": "Python에서 GIL이란?", "reference": "Global Interpreter Lock..."},
    {"question": "쿠버네티스의 Pod란?", "reference": "컨테이너의 최소 배포 단위..."},
    {"question": "NCCL AllReduce 설명해줘", "reference": "..."},
    # 도메인 특화 질문 50~100개
]

results = []
for tc in test_cases:
    # 평가 대상 모델에 질문
    model_response = call_your_model(tc["question"])
    
    # GPT-4로 평가
    score = judge_response(tc["question"], model_response, tc.get("reference"))
    score["question"] = tc["question"]
    score["model_response"] = model_response
    results.append(score)

# 결과 집계
import pandas as pd
df = pd.DataFrame(results)
print(df[["accuracy", "relevance", "completeness", "clarity", "safety", "overall"]].mean())
```

### 3. 키워드/패턴 기반 자동 검증 ###
```
# keyword_eval.py
import re

def keyword_check(response: str, expected_keywords: list) -> float:
    """응답에 핵심 키워드가 포함되어 있는지 체크"""
    found = sum(1 for kw in expected_keywords if kw.lower() in response.lower())
    return found / len(expected_keywords)

def hallucination_check(response: str, facts: dict) -> list:
    """알려진 사실과 대조하여 환각 감지"""
    errors = []
    for claim, truth in facts.items():
        if claim in response and truth not in response:
            errors.append(f"Hallucination: mentioned '{claim}' but wrong value")
    return errors

def safety_check(response: str) -> bool:
    """유해 콘텐츠 포함 여부"""
    dangerous_patterns = [
        r"sudo rm -rf /",
        r"DROP TABLE",
        r"<script>",
    ]
    for pattern in dangerous_patterns:
        if re.search(pattern, response, re.IGNORECASE):
            return False
    return True
```
